# Matter Protocol Reference Implementations

This document provides an overview of various reference implementations of the Matter protocol, detailing their functionalities, testing tools, and code examples categorized by implementation phase.

## Implementations

### 1. ESP-Matter
- **Description**: ESP-Matter is a Matter implementation on the ESP32 platform.
- **License**: Apache License 2.0.

### 2. Project CHIP TLV
- **Description**: This implementation provides a type-length-value (TLV) encoding for data serialization in Matter protocols.
- **License**: Apache License 2.0.

### 3. OpenThread SPAKE2
- **Description**: OpenThread's Secure Password Authenticated Key Exchange (SPAKE2) is used for secure communications.
- **License**: BSD License.

### 4. Matter Core Specification
- **Description**: This is the core implementation of the Matter protocol, specifying its standard behavior.
- **License**: Apache License 2.0.

## Testing Tools
- **chip-tool**: A command-line tool for testing Matter devices and protocols.
- **Wireshark**: Network protocol analyzer for troubleshooting and analysis of Matter communications.
